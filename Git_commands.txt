git clone clone-url  								- to check out

git branch 									- to know the current branch



git fetch									 -- to get all branches synchronized
git checkout branch-name							 -- to switch branch





To update after checking out and before committing:

git pull clone-url   								-to update like synchronize with repository


To commit:

git add -A             				 	      			 - select all


git commit -m "message"								 - to commit with message locally


git push clone-url								 - to push committed source to git.



To configure:

git config --global -l 								- to list the config file

git config --global -user.name "BT_Developer" 					- to set user name
git config --global -user.mail "sanjay@gmail.com"				- to set user mail
